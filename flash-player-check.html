Check Flash Player

If you see a blank red box above, Flash Player is not installed or enabled.
Install it from here.

If you see a Flash Player information icon, your Flash Player is timebombed.
Patch your Flash Player using this tool.

If you see your Flash Player version, you're good to go!

Note that most modern browsers don't support Flash Player anymore.
Compatible browsers for Windows include Internet Explorer, Waterfox, K-Meleon, Pale Moon, and Basilisk.
Waterfox is recommended for Linux or Mac users. Pale Moon and Basilisk also have Linux versions available.
